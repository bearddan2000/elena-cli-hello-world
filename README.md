# elena-cli-hello-world

## Description
A POC for elena programming language.

## Tech stack
- g++
- git
- make

## Docker stack
- ubuntu:22.04
- rzuckerm/elena

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
